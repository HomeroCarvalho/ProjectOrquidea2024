◆ LoadCodeFromHeaders() [1/2]

void parser.Classificador.LoadCodeFromHeaders ( List< string > tokens)

processamento de nomes de classes, metodos, propriedades, operadores.

1- nao le classes orquidea base, se ja estão nos elementos do classificador. se não estão, faz o processamento. 2- remove classes de mesmo nome, anteriores, armazenada nos elementos do classificador. 3- le a classes dos headers vindo do cenario de teste currente, e dá processamento, armazenando no classificador.

Parameters
tokenslista de tokens contendo definicoes de classes.
