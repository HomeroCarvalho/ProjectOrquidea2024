Parameters

escopocontexto onde a instrucao está.

EXPRESSOES: a lista de expressões da instrução foi feita na seguinte sequência: 0- NOME "create" 1- tipo do objeto 2- nome do objeto 3- tipo template do objeto: Objeto/Vetor. 4- expressoes indices vetor. 5- expressoes parametros. 6- indice do construtor. 7- nome do objeto caller.

ID ID = create ( ID , ID ) --> exemplo: int m= create(1,1).
